# Productivity Tracker – Project Statement

## 1. Problem Statement
Managing weekly tasks, meeting deadlines, and staying productive can be challenging for many individuals. Without a proper tracking system, tasks may be forgotten, completed late, or improperly prioritised. This leads to reduced productivity and poor time management.  
This project aims to provide a simple, Python-based tool that helps the user track tasks, analyse productivity, and receive helpful feedback for improvement.

## 2. Scope of the Project
The scope of this project includes:
- Collecting weekly tasks from the user
- Storing task details such as name, deadline, and priority
- Marking tasks as completed
- Tracking time spent on tasks and other activities
- Calculating the user’s productivity percentage
- Providing a productivity rating and improvement tips
- Displaying a weekly summary report with colour-coded formatting

Out of scope:
- Graphical user interface (GUI)
- Integration with databases
- Cross-platform mobile or web interface
- Long-term data storage

## 3. Target Users
This project is designed for:
- Students managing academic and personal tasks
- Working professionals tracking weekly goals
- Anyone who needs weekly task planning and progress analysis

The tool is beginner-friendly and accessible to all types of users.

## 4. High-Level Features
The key features of the Productivity Tracker are:

### Task Management  
- Add weekly tasks  
- Include task name, deadline, priority  
- Mark completed tasks  

### Automatic Sorting  
- Tasks are sorted by deadline for better visibility  

### Productivity Calculation  
- Calculates productivity as:  
  *(tasks completed / total tasks) × 100*

### Ratings & Colour Coding  
- Provides colour-based ratings using Colorama  
- High, Medium, and Low priority tasks are colour-coded  
- Tips and motivational messages based on performance  

### Weekly Summary Report  
- Displays total tasks, completed tasks, remaining tasks  
- Shows productivity percentage, tips, ratings  
- Colour-coded task list for easy interpretation  

### Modular Design  
- Organized into multiple modules for easy maintenance  
- Implements clean programming practices  

## 5. Tools & Technologies Used
- **Python 3.x**
- **Colorama** for terminal colour formatting
- **VS Code** for development

## 6. Outcome
The Productivity Tracker provides a structured approach to weekly task management.  
It helps users stay organised, improve productivity, and receive constructive feedback through a simple but effective command-line interface.
